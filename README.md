# 🌍 Wanderlust Travel Blog

Sitio web moderno y visualmente impactante diseñado para documentar experiencias de viaje.

## 🚀 Características
- **Diseño Ultra-Limpio**: Inspirado en revistas de fotografía premium.
- **Instagram Ready**: Integración directa de posts y reels mediante embeds.
- **Totalmente Responsivo**: Experiencia fluida en móviles y tablets.
- **Interacciones Pro**: Animaciones suaves al hacer scroll y efectos hover avanzados.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Tailwind CSS)
- JavaScript Vanilla
- Google Fonts (Playfair Display & Inter)

## 📦 Instalación
Solo clona este repositorio y abre `index.html` en tu navegador.
